.flake8 기능 

1. .flake8 개요
pdfminer.six의 Linting 규칙을 커스터마이징한 파일

2. Linting이란?
코드 유지보수를 위한 잠재적인 에러를 방지하기 위해 어떤 규칙이 지켜지지 않았을 때
해당 부분을 설정한 옵션에 맞게 검사해주고 교정해주는 일련의 과정을 말한다.
예를 들어 함수 뒤에 띄어쓰기를 안했다면 강제로 띄어쓰기를 넣어주거나 삭제해주거나 띄어쓰기가 안되어있다고 알려주는 등의 일을 하면서 코드의 포맷을 일정하게 잡아준다.

3. max-line-length = 88
코드의 라인 당 글자 수가 최대 88로 정해져있다.

4. extend-ignore = E203 : linting중 무시하고자 하는 규칙을 ignore로 설정할 수 있는데 그 ignore을 확장한 것에 대한 코드이다.
'E203: False positive "whitespace before ':' " on list slice.'인 오류, 즉 리스트의 ':'앞에 공백이 생길 때 발생하는 오류를 무시하도록 설정하고 있다.

5. gz파일
데이터를 압축하기 위해 사용되는 파일 형식 중 하나이다. "gzip"라는 프로그램으로 압축된 파일을 가리키며, 주로 리눅스와 유닉스 기반 시스템에서 사용된다.

6. pyc파일
파이썬 소스 코드 파일을 파이썬 인터프리터가 실행하기 전에 바이트 코드로 컴파일한 파일이다. 파이썬 소스코드를 실행할 때 더 빠른 실행을 가능하게 한다.
